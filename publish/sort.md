quickSort 알고리즘에서 [[timsort]] 알고리즘으로 바뀜.

```js
  

const num = [1, 5, 2, 6, 7, 89, 9];

num.sort((a, b) => {

console.log(a, b);

});

  

console.log(num); 
// 5 1 
// 2 5 

// a는 비교하려는 숫자
// b는 비교대상 
// 버블 정렬 
```

핵심 : 양수와 음수
